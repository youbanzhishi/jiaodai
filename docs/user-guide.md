# 胶带 用户指南

## 什么是胶带？

胶带是一个时间封存平台，让你可以"现在写，条件到了才开"。

就像用胶带封住的信封——封口有签名（hash上链），撕开就知道有没有人动过。

## 如何使用

### 创建封存

1. 写下你想封存的内容（文字/图片/视频）
2. 选择解封条件：
   - **心跳失联**：指定确认人，确认人N天未确认则触发解封
   - **双向匹配**：只有对方也封存了，双方才同时收到通知
   - **指定日期**：到指定时间自动解封
   - **多人确认**：N人中M人确认后解封
3. 选择查看人
4. 封存！

### 查看封存

- 查看封存凭证（证明内容未被篡改）
- 查看封存状态（是否已解封）
- 解封后查看内容（需验证身份）

## 常见场景

### 暗恋表白

1. 输入对方手机号创建封存
2. 如果对方未注册，对方会收到短信邀请
3. 只有对方也对你创建封存，双方才会同时收到通知
4. 只有一方 → 永远静默，对方不会知道

### 遗嘱交代

1. 实名认证后创建遗嘱封存
2. 指定确认人（家人/朋友）
3. 定期给确认人发送确认消息
4. 确认人超时未确认 → 进入缓冲期 → 最终人工确认 → 解封

### 时间胶囊

1. 写下给未来的话
2. 设定解封日期
3. 到时间自动解封
4. 指定查看人（或自己）
